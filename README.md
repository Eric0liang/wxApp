# wxApp

## 项目介绍
wxApp是一个由gulp构建微信小程序项目

### 特点
- 支持微信小程序原生目录结构
- 支持postcss预处理器
- 支持编译HTML模板成wxml
- 支持图片资源的编译

## 使用说明

### 安装
```shell
$ git clone git@github.com:lijiliang/wxApp.git
$ cd wxApp
$ npm install
$ npm i gulp -g
```

### 开发模式
```shell
npm run dev
```

### 构建模式
```shell
npm run build
```
